指標相關的評比建議

根據不同的環保指標，例如二氧化碳密度、廢棄物密度和人力發展等項目，我們提供具體的建議，幫助企業達成更好的表現。
分數過低可能會被認為有漂綠嫌疑，而較高分則顯示企業更接近永續經營。

分數來源:

我們建立總永續發展指標 (SDI: Sustainability Development Index) 作為判斷標準。

計算方法和評分公式:

  • (E) 環境: 採用溫室氣體排放密集度與廢棄物密集度作為衡量環境管理的主要指標。
  • (S) 社會: 以員工薪資平均數與管理職女性占比強調公司在人力資源發展和多元平等上的表現。
  • (G) 政府治理: 以管理職女性占比作為治理平等的核心衡量。
  • (B) ISO 14001 認證加分: 將這份國際認證納入公式，為有經此認證之企業額外加分。若通過 ISO 14001 認證，則 B = 1（滿分），未通過則 B = 0。

設計權重分配:

  • (E) 環境: 採用溫室氣體排放密集度與廢棄物密集度作為衡量環境管理的主要指標。
  • (S) 社會: 以員工薪資平均數與管理職女性占比強調公司在人力資源發展和多元平等上的表現。
  • (G) 政府治理: 以管理職女性占比作為治理平等的核心衡量。
  • (B) ISO 14001 認證加分: 將這份國際認證納入公式，為有經此認證之企業額外加分。若通過 ISO 14001 認證，則 B = 1（滿分），未通過則 B = 0。

評分公式：

(E) 環境 = 1 – [(溫室氣體排放密集度 + 廢棄物密集度) / 最大值]

評分公式：

(S) 社會 = [員工薪資數 + (管理職女性比 * 1000)] / 最大值

評分公式：

(G) 政府治理 = 管理職女性占比 / 最大值

💫💥 綜合評分計算

將三個指標的得分進行加權平均，再根據是否有通過 ISO14001 當作加分項，生成最終的綠色信用評分。

公式：

SDI = (E_w．E + S_w．S + G_w．G + B_w．B) / (E_w + S_w + G_w + B_w) * 100

5. 評分範圍與解釋

根據企業的綠色信用分數，我們將企業分為不同的評分級別，以便更直觀地了解其環保表現。

評分級別 分數範圍 含義
A+ 90 - 100 表現卓越，具有領導地位的環保企業
A 80 - 89 環保表現優秀，符合高標準
B 70 - 79 符合環保標準，有進步空間，但可能有漂綠嫌疑
C 50 - 69 基本符合標準，但需進一步改進
D 0 - 49 環保表現不足，需要大幅改善
